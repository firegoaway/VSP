Назначение утилиты.

Программа GroupF2Rx предназначена для расчета таблиц толщин огнезащитного покрытия  по уравнению Фурье для обеспечения пределов огнестойкости по признаку  R с использованием  результатов  экспериментальных данных  огневых испытаний двутавров по  ГОСТ Р 53295.

Описание расчета - приложение 2 "Проект НД для Rx3" .
Алгоритм расчета соответствует  приложению  F ENV 13381-4  в части использования λchar.


Подготовка данных для программы.

Экспериментальные данные  и параметры  расчета хранятся в файле формата sqlite3 с расширение *.fpm2sqlite. Результаты расчета  не сохраняются.

Добавление экспериментальных данных осуществляется  из файла формата csv.

Расчет толщин для Rx3

Расчет толщин покрытия для выбранных пределов огнестойкости при указанном диапазоне приведенных толщин  и нормированных критических температурах.




Вкладка "Исходные данные".

1.    Поле для отображения  образцов.  Для   расчетов  используются образцы отмеченные галочкой.

2.    Параметры  обработки экспериментальных данных.  Параметры  обработки  распространяются на все образцы.
 
3.  Данные образца. Показатели образца доступны для редактирования.

4.  Коэффициент коррекции для учета влияния нагрузок на устойчивость огнезащитного покрытия к разрушению при воздействии высоких температур. Значение в диапазоне 1.0-1.1.

5.  Добавление  образца в формате csv, удаление выбранного образца.


Вкладка "Коэффициент теплопроводности".





1.    Результаты расчета λ(θp) .

2.    Результаты расчета   λm и  λave.  

4.    Результаты расчета λave в  выбранных интервалах толщин.


Вкладка "Проверка".




1.    Время достижения нормированных критических температур по экспериментальным данным.

2.    Время достижения нормированных критических температур по расчетным данным.

3.    Разница между экспериментальными и расчетными данными по времени достижения нормированных критических температур.

4.    Результаты расчета λchar.

5.    Переключатель для выбора набора коэффциентов теплопроводности покрытия для расчета.

6.   Критерии для оценки приемлемости результатов расчета.

Рекомендуемые значения критериев по  ENV 13381-4:
- положительная разница по времени достижения для каждой заданной температуры каждого образца не должен превышать результат испытаний более чем на 30 %;
- сумма значений разниц по времени достижения всех заданных температур для всех образцов должна быть меньше нуля;
- количество положительных значений разниц для всех заданных температур всех образцов не должна превышать 20 % от всего количества значений.

7.  Наименьший коэффициент коррекции набора коэффциентов теплопроводности для достижения заданных критериев:
- для λchar -  больше или равно 0;
- для  λave(d,θp) больше или равно 1 .

 
Вкладка «Расчет толщин для Rx3».

 
1.    Исходные данные для расчета толщин.
2.    Выбор требуемых пределов огнестойкости.
3.    Оценка времени процесса расчета.
4.    Результаты расчета.

Выбор коэффициентов теплопроводности покрытия для расчета толщины покрытия  на вкладке «Проверка».

Допуски для экстраполяции по  ENV 13381-4  при использования λchar.

по приведенной толщине

-20% от минимального экспериментального значения

 +50% от максимального экспериментального значения значения

по толщине покрытия

-20% от минимального экспериментального значения

 +20% от максимального экспериментального значения значения

 

Дополнительная информация по программе.

Результаты расчета могут быть скопированы в буфер обмена для последующего использования.
