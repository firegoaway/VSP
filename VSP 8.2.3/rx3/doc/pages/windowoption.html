Окно настройка

Данные по настройкам вкладок "Значение по умолчанию", "Таблица конструкция", "Таблица Раздел"  сохраняются в файле "rx3.xml".  При переходе на новую версию программы  для сохранения настроек можно скопировать файл "rx3.xml"  старой версии в папку с новой версией.
Если файл удалить, он создается автоматически при запуске программы.


Вкладка «Огнезащитный материал»

Предназначено для добавление и удаление огнезащитного материала.
Данные по огнезащитному материалу сформированы в файле с расширением sfmp. Каждому материалу присваивается свой числовой идентификатор.
Программа различает огнезащитный материалы по числовому индентификатору.
Файл огнезащитного материала формирует утилитой Tfpm.create.
При формировании файла необходимо использовать  данные для пределов огнестойкости.
Данные  для групп огнезащитной эффективности могут быть пересчитаны с помощью утилит Group2Rx,  GroupF2Rx, VniipoF2Rx.

Вкладка «Значения по умолчанию»

Можно выбрать значение показателей,  присваеваемых  конструкции при добавлении.
 
Вкладка «Таблица Конструкция»

Настройка видимых показателей активной  таблицы «Конструкции».
Внешний вид таблицы "Конструкция" настраивается.
Для настройки - открыть окно "Настройка"  вкладка  "Таблица Конструкция".
Открыть вкладку также можно через всплывающее меню - правая кнопка мыши на активной таблице.

Вкладка «Таблица Раздел».

Настройка видимых показателей  активной таблице «Раздел».
Внешний вид таблицы "Раздел" настраивается.
Для настройки - открыть окно "Настройка"  вкладка  "Таблица Раздел".
Открыть вкладку также  можно через всплывающее меню - правая кнопка мыши на активной таблице.
