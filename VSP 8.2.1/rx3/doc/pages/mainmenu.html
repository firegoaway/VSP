Главное меню

Файл
    Новый  Создает новый файл.
    Открыть  Открывает данные из выбранного файла
    Добавить Добавляет данные из выбранного файла к  существующим
    Сохранить  Сохраняет данные в файле
    Сохранить как Сохраняет данные в файле с другим именем
    Выход Закрывает программу

Правка
    Копировать Копирует данные выбранной  конструкции в память программы
    Вставить Добавляет данные конструкции из памяти программы  к выбранному разделу
    Удалить Удаляет выбранную конструкцию или раздел  с конструкциями
    Свернуть все Скрывает конструкции во всех разделах
    Настройка Открывает окно настройки программы

Расчет
    Предел огнестойкости Открывает окно для  расчета предела огнестойкости  выбранной конструкции
    Выбрать/изменить огнезащитный материал Открывает окно выбора или замены огнезащитного материала для выбранного раздела или всех разделов.

Экспорт
    Копировать в буфер обмена Копирует  таблицу в буфер обмена Windows
    Сохранить как CSV Сохраняет  таблицу в файле формата CSV
    Открыть в браузере Открывает таблицу в браузере установленном по умолчанию

Информация
    Справка Открывает справку программы
    Проект НД по расчету Открывает pdf файл с проектом инструкции
по расчету
    Справочные данные Открывает окно со справочными данными
    О программе Открывает окно c  информацией  о программе
