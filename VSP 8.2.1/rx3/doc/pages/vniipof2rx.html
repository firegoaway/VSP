Назначение утилиты.

Программа VniipoF2Rx предназначена для расчета толщины огнезащитного вспучивающегося покрытия для обеспечения требуемого предела огнестойкости по методике ВНИИПО (поз.№3).

Вкладка "Исходные данные".

 

1.    Поле для отображения  образцов.  Для   расчетов  используются образцы отмеченные галочкой.

 2.    Параметры  обработки экспериментальных данных.  Параметры сглаживания распространяются на все образцы.
 
 3.    Данные образца. Показатели образца доступны для редактирования.

 4.    Добавление  образца в формате csv, удаление образца.

Вкладка "Коэффициент теплопроводности".

1.    Параметры приведения λ(t) к λ(θs). Можно сгладить результаты расчета λave .

 2.    Результаты расчета λ(θs) и λave .

 3.    Результаты расчета   λ123 в диапазоне 100-750 0 С с интервалом 50 0 С.

 4.    Наборы коэффициентов теплопроводности покрытия для  расчетов толщин.



Вкладка "Проверка".

1.    Время достижения нормированных критических температур по экспериментальным данным.

 2.    Время достижения нормированных критических температур по расчетным данным.

 3.    Разница между экспериментальными и расчетными данными по времени достижения нормированных критических температур.

 4.    Результаты которые могут быть использованы как критерии.

 5.     Переключатель для выбора набора коэффциентов теплопроводности покрытия для расчета.


Методика ВНИИПО не предусматривает проверку результатов расчета. Вкладка в большей части является информационной.

 
При расчете используются средние по всем образцам значения  плотности стали, удельной теплоемкости стали, удельной теплоемкости покрытия, плотности покрытия, степени черноты.

 2.    Графическое отображение расчетных кривых нагрева.

 3.    Расчетные данные кривых нагрева.

Выбор коэффициентов теплопроводности для расчета толщины покрытия на вкладке «Проверка».

При расчете используются средние по всем образцам значения  плотности стали, удельной теплоемкости стали, удельной теплоемкости покрытия, плотности покрытия, степени черноты.

Вкладка «Расчет толщин для Rx3».


 
1.    Исходные данные для расчета.

 2.    Выбор требуемых пределов огнестойкости.

 3.    Оценка времени процесса расчета.

 4.    Результаты расчета.

Выбор коэффициентов теплопроводности покрытия для расчета толщины покрытия  на вкладке «Проверка».

В методике ВНИИПО не установлены однозначные границы применимости результатов расчета по приведенной толщине и толщине покрытия.

При расчете используются средние по всем образцам значения  плотности стали, удельной теплоемкости стали, удельной теплоемкости покрытия, плотности покрытия, степени черноты.
 
Дополнительная информация по программе.

Результаты расчета могут быть скопированы в буфер обмена для последующего использования.
